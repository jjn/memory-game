# ❄️ Frozen Memory Game

A fun, Frozen-themed memory card matching game built for little ones!

Flip the cards and find all the matching pairs of your favorite Frozen characters: Elsa, Anna, Olaf, Sven, Kristoff, and Bruni.

## How to Play

1. Click or tap a card to flip it over.
2. Flip a second card to find a match.
3. If the cards match, they stay face-up. If not, they flip back.
4. Find all 6 pairs to win!

## Running Locally

You need a local web server to run the game. Here are a few options:

### Python (recommended — usually pre-installed on macOS/Linux)

```bash
# Python 3
python3 -m http.server 8000

# Then open http://localhost:8000 in your browser
```

### Node.js

```bash
npx serve .

# Then open the URL shown in the terminal
```

### PHP

```bash
php -S localhost:8000

# Then open http://localhost:8000 in your browser
```
